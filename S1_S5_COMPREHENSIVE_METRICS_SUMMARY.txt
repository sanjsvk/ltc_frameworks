SECTION 7: KEY FINDINGS SUMMARY (FOR PAPER)
────────────────────────────────────────────

FINDING 1: bsts 1.02x Ratio — Structural Robustness Centrepiece
  → Only model with near-invariant error distribution across smooth (S1) and paused (S2) spend
  → 82.4% (S1) → 81.0% (S2) → 76.8% (S3) → 81.6% (S4) = most consistent trajectory
  → Paper centrepiece: "Bayesian Structural Time-Series maintains structural robustness comparable to DGP"

FINDING 2: F2 Paradox — Ratio <1.0 Reflects Overfitting, Not Robustness
  → finite_dl (0.69x) and koyck (0.77x) show error improvements in pause window
  → But channel-level attribution analysis reveals systematic misattribution
  → False robustness: baseline overfitting correction, not structural robustness

FINDING 3: Robustness Spectrum Taxonomy (Three Tiers)
  → Tier 1 (Gold): 0.95–1.10x — bsts (1.02x S2), mcmc_stock (0.93–0.96x)
  → Tier 2 (Silver): 1.10–1.35x — kalman_dlm (1.35–1.42x), finite_dl (1.05–1.15x)
  → Tier 3 (Fragile): >1.35x — geo_adstock (1.41–1.51x), weibull (1.49x)

FINDING 4: Aggregate vs Channel Validation (Methodological Contribution)
  → ARDL: 68.8% aggregate recovery + 0% per-channel recovery (offsetting errors)
  → Koyck: 43.0% aggregate + inverted channel ranking (59.3% to Display vs 2.2% to TV)
  → CRITICAL: Aggregate accuracy does not validate channel-level precision

FINDING 5: geo_adstock vs Kalman_dlm Ratio Coincidence (1.41x Both S2)
  → Identical ratio masks fundamentally different error structures
  → geo_adstock: Volatile across scenarios (±13–40pp range)
  → kalman_dlm: Stable with predictable degradation (±1–11pp range)
  → Cross-scenario consistency reveals true robustness

FINDING 6: MCMC Non-Monotonic Trajectory (S1→S2→S3→S4)
  → 72.6% → 61.4% → 98.8% → 91.0% — unique non-monotonic pattern
  → S3 peak: Seasonal signal provides additional identification source
  → Recommendation: MCMC optimal for businesses with strong seasonality

FINDING 7: ARDL Instability (Temporary vs Permanent Breaks)
  → S2 (temporary): 68.8% recovery ← spend pause improves identification
  → S4 (permanent): -19.8% recovery ← permanent shift causes sign-flip
  → CRITICAL: Models robust to temporary pauses may catastrophically fail on permanent shifts

FINDING 8: S5 Universal Collapse (Signal Threshold Requirement)
  → All 10 models: 0% recovery when LTC signal halved
  → Framework hierarchy disappears (F3 = F2 = F1 at 0%)
  → FINDING: Below signal threshold, method choice becomes irrelevant

FINDING 9: Cross-Scenario Stability as Robustness Metric
  → Single-scenario ratio insufficient; cross-scenario StdDev more informative
  → bsts: 2.4pp StdDev (most stable); ardl: 38.9pp StdDev (most volatile)
  → Proposed metric: Robustness Score = Mean / (1 + StdDev)

OVERALL CONCLUSIONS

1. NO UNIVERSAL WINNER — Framework effectiveness is scenario-dependent
   - F3 dominates S1/S2/S4 (78.4% avg)
   - mcmc_stock peaks on S3 seasonality (98.8%)
   - All collapse on S5 weak signal (0%)

2. IDENTIFICATION VULNERABILITY CONFIRMED
   - Static models (F1) fail when spend variation obscured (S3 collinearity)
   - Dynamic models (F2) unstable on permanent vs temporary shifts (ardl S2→S4)
   - State-space models (F3) robust across patterns but calibration-sensitive (S5)

3. SINGLE METRICS ARE INSUFFICIENT
   - Pause-window ratio is necessary but not sufficient
   - Aggregate accuracy masks channel-level failures (ardl, koyck cases)
   - Cross-scenario consistency reveals true robustness (bsts 2.4pp vs ardl 38.9pp)

4. PRACTITIONER RECOMMENDATIONS
   - For production: F3 + bsts (78.6 robustness-adjusted score)
   - For seasonality: mcmc_stock (98.8% on high collinearity)
   - For backup: F2 models (42.8% avg, more stable than F1)
   - For caution: F1 models (should not use without structural break detection)

5. SIGNAL STRENGTH IS A BOUNDARY CONDITION
   - Models require minimum LTC signal-to-noise for identification
   - S5 demonstrates that below threshold, framework choice becomes irrelevant
   - Practitioners must assess signal strength before model selection
